Img tag of size 200px by 300px

Notice how the image is stretched. When using width and height with an img tag, it stretches it to the width and height specified.

Background image of size 200px by 300px

Instead of using an img tag, you can create an empty div with a set width, set height, and a background image. By using the background-size css property, and the other background properties, you can make a nice photo that fills the size and is centered, and is not stretched.
